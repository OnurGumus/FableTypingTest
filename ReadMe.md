## Build steps

Clone the project and in the root folder

``` bash
cd src/Client
cd npm run start
```

Then navigate to 

```
http://localhost:8080
```


## To build for production 

``` bash
cd src/Client
cd npm run build
```

deploy the `src/Client/dist` folder
